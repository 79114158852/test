Внедрение зависит от выбранных подходов и архитектуры проекта, возможно потребуется добавить паттерн репозиториев и/или подкрутить решение под один из модульных пакетов Laravel. Так же, возможно, используется какой-либо пакет генерации апи документации, что потребует дополнительных манипуляций. К тому же, нельзя исключать вероятность микросервисной архитектуры, поэтому трудно отвечать на вопрос о внедрении в проект, о котором ничего не известно.
